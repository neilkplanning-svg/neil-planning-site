# Neil Kappel - אתר תכנון פיננסי

אתר מקצועי לשירותי תכנון פיננסי למשפחות.

## מבנה האתר

```
neil-planning-site-main/
├── index.html           # דף הבית
├── about.html           # אודות ניל
├── knowledge.html       # מרכז ידע
├── resources.html       # מקורות מידע
├── style.css            # עיצוב האתר
├── script.js            # פונקציונליות JavaScript
├── sitemap.xml          # מפת אתר לגוגל
├── robots.txt           # הוראות לבוטים
├── SETUP_INSTRUCTIONS.md # הוראות הגדרה מפורטות
├── tools/
│   ├── tool-compound.html      # מחשבון ריבית דריבית
│   ├── tool-loan.html          # מחשבון הלוואה
│   ├── tool-leverage.html      # מחשבון מינוף (LTV)
│   ├── tool-refinance.html     # כדאיות מחזור
│   └── tool-invest-leverage.html # כדאיות מינוף להשקעה
└── articles/
    └── article-basics.html     # מאמר יסודות הכלכלה
```

## טכנולוגיות

- HTML5 Semantic
- CSS3 (CSS Variables, Flexbox, Grid)
- JavaScript ES6+
- Chart.js לגרפים
- Font Awesome לאיקונים
- Google Fonts (Heebo, Montserrat)

## התקנה מהירה

1. העלו את הקבצים לשרת
2. עדכנו את `GA_MEASUREMENT_ID` ב-HTML files עם ה-ID מ-Google Analytics
3. הגדירו SSL certificate
4. הגישו sitemap ל-Google Search Console
